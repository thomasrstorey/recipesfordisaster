# recipeaterbot

automatically generate impossible recipes

## Getting Started

```javascript
var recipe = '';
var recipeaterbot = require('recipeaterbot.js');
recipe =  recipeaterbot.generateIngredientsList() +
       recipeaterbot.generateStepsList();
console.log(recipe);
```
